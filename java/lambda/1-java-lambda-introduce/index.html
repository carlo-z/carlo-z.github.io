3、串行 -> 并行

Java 8 支持对集合的并行处理，而 lambda表达式 是”提供这种支持“ 必备的一环；

原理：

  1. 将大任务 递归的 切分为 ”足够小“ 的 小任务（能串行执行为止）；
  2. 使用 cpu 的多核 分别执行完以后；
  3. 将结果 结合起来 返回；
