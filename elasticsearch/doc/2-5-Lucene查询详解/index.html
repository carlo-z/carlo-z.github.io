2.5.3、词项搜索（TermQuery）

TermQuery 是 最常用的 Query

TermQuery 是 Lucene中搜索的最基本单位

本质上：一个词条就是一个 key/value 对

使用TermQuery：

  • 首先构造一个 Term对象；Term term = new Term(“title”, “美国”);
  • 然后使用Term对象为参数，构造一个TermQuery对象；TermQuery query = new TermQuery(term);
